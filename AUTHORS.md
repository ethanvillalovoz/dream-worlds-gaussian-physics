# Authors and contributions

Dream Worlds was developed as a Spring 2026 final project for Georgia Tech's CS 8803: Computer Graphics in the AI Era.

## Original project authors

### Michael Walker

- Set up pretrained Gaussian rendering.
- Integrated the packaged Gaussian Splatting implementation.
- Developed the shared experimental framework.
- Implemented the uniform-gravity and randomized inverse-mass baselines.
- Produced the combined comparison video.

### Ethan Villalovoz

- Implemented the spatially varying wind-field experiment.
- Implemented the low- and high-wind ablation variants.
- Organized and documented the repository.
- Extracted representative result frames.
- Led writing and assembly of the final technical report.

## Maintained fork

This fork is maintained by Ethan Villalovoz. Repository presentation, documentation, citation metadata, provenance notes, and static validation added after the course submission do not change the authorship of the original project, experiments, paper, or results.

The canonical source history is preserved through the GitHub fork relationship to [`MWalkGATech/CGAI-Final-Project-Dream-Worlds`](https://github.com/MWalkGATech/CGAI-Final-Project-Dream-Worlds).
