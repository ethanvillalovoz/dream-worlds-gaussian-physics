# Changelog

## [1.0.0] - 2026-07-21

First public research and portfolio release of Dream Worlds.

- Preserves the five original Gaussian Splatting motion experiments and cleared course notebook.
- Documents the CUDA environment, external Ficus checkpoint, output paths, and reproducibility limitations.
- Includes the technical report, qualitative result strips, labeled motion comparison, and conceptual method diagram.
- Adds repository validation, citation metadata, authorship records, contribution guidance, and asset provenance.
- Licenses the jointly authored project code under the MIT License while retaining separate terms for papers, media, data, pretrained assets, and third-party components.

This release packages and documents the completed course-project evidence; it does not claim new experiments beyond the archived submission.
